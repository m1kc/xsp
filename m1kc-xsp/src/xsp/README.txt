Заметки на полях.
-----------------

Обозначения:
Look - бинарный или текстовый
Type - уже собственно тип

Как добавить поддержку нового типа пакета:
1. Прописать константу в XSPConstants;
2. Прописать константу и метод в Session;
3. Добавить метод в UIProxy;
4. Добавить метод в наследник UIProxy (SessionFrame).
